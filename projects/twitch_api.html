Twitch API

API Experiment

A site built to experiment with the Twitch TV API for potential use in a future project.

The main purpose of this site was to test the waters for potentially using this API in a future project. My main goal here was first to send a get request to the API to ensure I was receiving the data I needed to access. Once this data was available to me I had to append it to the HTML in an aesthetically pleasing, usable way. This meant the user could easily see the data and understand what each value was showing them without the need for excessively heading each piece of information.

I have included a number of streams to ensure there should always be a mix of states showing off all features of the API that I have utilised.
