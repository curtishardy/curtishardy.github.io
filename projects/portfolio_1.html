Portfolio

This project is a portfolio I created to use Bootstrap and function as a portfolio.

With this project I wanted to showcase my ability to utilise Bootstrap and customise it to suit my needs when required. I also wanted all of the functionality that would be required from a true live site to be there.

This meant I required the site to load quickly and provide all of the information you'd expect to see on a portfolio in a clear, concise fashion. It would also mean any projects would need their own page, much like this one, to explain the project, the thought process behind it and also display some screenshots of the site. All of this had to be thought through and planned for in a responsive way, to ensure users had access to everything regardless of their device.
