Portfolio

This project is a portfolio I created to use Foundation and function as a portfolio.

Using this as an opportunity to showcase my ability to utilise Foundation and tailor it to my needs, along with using it as a way to think through the way a site needs to be designed to function best for the user.

This meant that I had to think the design through due to needing all information to be clearly visible to the user no matter what device they were using to view the site. It also meant I wanted to try and keep the site to a single page to minimise the chance of the user missing information tucked away in some corner of the site they were unable to find. Due to wanting to keep it on a single page I had to utilise a part of Foundation to create a slider that allowed the user to slide through my projects and provide them with details about the project they were viewing.
